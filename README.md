# 新品空运优先级计算器

用于首批测款时，按照“每 1 元空运增量成本能够换回多少预期提前验证价值”排列 SKU 的空运优先级。

## 核心指标

```
AAI（空运加速指数）= 预期提前价值 ÷（空运成本 − 海运机会成本）
```

测试采购价只统计首批采购预算，不直接进入 AAI。未来贡献利润使用正常大货采购价计算。

## 功能

- 计算实重、体积重、计费重和抛重倍数
- 支持 40HQ 体积/重量占柜机会成本
- 支持成功率、销量、正常大货贡献利润和有效提前天数
- 自动输出 AAI 排名、空运/海运建议及预算内组合
- 支持 JSON 导入导出、CSV 结果导出、打印和本机自动保存
- 单文件 HTML，无后端，录入数据不会上传服务器
