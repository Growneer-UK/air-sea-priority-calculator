# 新品空运优先级计算器

用于首批测款时，按照“成功情景下，每 1 元空运增量成本能够提前释放多少毛利潜力”排列 SKU 的空运优先级。

## 产品点口径

- 1 产品点 = 产品成功后年毛利潜力 ¥120,000
- 允许填写小数，例如 0.5 点代表年毛利 ¥60,000
- 产品点必须使用公司统一的毛利定义

## 核心公式

```text
年毛利潜力 = 产品点 × ¥120,000
提前毛利潜力 = 年毛利潜力 × 有效提前天数 ÷ 365
AAI = 提前毛利潜力 ÷（空运成本 − 海运机会成本）
```

AAI 是成功情景下的优先级指标，不是经过成功率加权的财务收益率。空运和海运不改变同一产品的成功概率，因此主排名不使用产品级成功率；如需组合期望口径，可用统一的整体测款成功率校准最终选择线，统一乘数不会改变排名。

## 功能

- 输入每个 SKU 的产品点和有效提前天数
- 计算实重、体积重、计费重和抛重倍数
- 支持 40HQ 体积/重量占柜机会成本
- 输出 AAI 排名、成功情景盈亏平衡点数和提前天数
- 按空运增量预算或最多 SKU 数给出组合
- 支持 JSON 导入导出、CSV 结果导出、打印和本机自动保存
- 单文件 HTML，无后端，录入数据不会上传服务器
