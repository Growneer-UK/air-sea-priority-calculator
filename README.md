# 新品空运优先级计算器

用于首批测款时，按照“成功情景下，每 1 元空运增量成本能够提前释放多少毛利潜力”排列 SKU 的空运优先级。

## 产品点口径

- 1 产品点 = 产品成功后年毛利潜力 ¥120,000
- 新增产品默认 0.1 点，即年毛利潜力 ¥12,000，可逐个修改
- 允许填写小数，例如 0.5 点代表年毛利 ¥60,000
- 产品点必须使用公司统一的毛利定义

## 核心公式

```text
年毛利潜力 = 产品点 × ¥120,000
提前毛利潜力 = 年毛利潜力 × 有效提前天数 ÷ 365
AAI = 提前毛利潜力 ÷（空运成本 − 海运机会成本）
```

AAI 是成功情景下的优先级指标，不是经过成功率加权的财务收益率。空运和海运不改变同一产品的成功概率，因此主排名不使用产品级成功率；如需组合期望口径，可用统一的整体测款成功率校准最终选择线，统一乘数不会改变排名。

## 默认空海运边界

默认产品点为 0.1、有效提前天数为 35 天：

```text
每提前 1 天的毛利潜力 = ¥12,000 ÷ 365 = ¥32.88
35 天提前毛利潜力 = ¥12,000 × 35 ÷ 365 = ¥1,150.68
```

因此在最低入选线保持 AAI=1、预算足够且没有强制运输规则时：

- 空运增量成本 ≤ ¥1,150.68：空运
- 空运增量成本 > ¥1,150.68：海运

这里的空运增量成本是“空运成本 − 海运机会成本”。该边界是产品成功情景口径；若要使用统一成功率做严格期望值判断，则将 ¥1,150.68 再乘以公司统一测款成功率。

重量和体积通过运输成本进入边界：

```text
体积重 = 外箱长 × 宽 × 高 ÷ 抛重除数
空运计费重 = MAX（实重，体积重），再按货代规则进位
空运边界计费重 =（海运成本 + 提前毛利潜力 − 空运固定费）÷ 空运单价
```

以默认空运费 ¥40/kg 为例，若暂不计海运机会成本和空运固定费，0.1 点、提前 35 天对应的整批边界计费重约为 28.77kg。实际计费重不超过边界时达到 AAI≥1；抛货因为体积重高，更容易落到海运一侧。

页面提供独立的“边界计费重快速计算”模块，可输入产品点、有效提前天数、海运机会成本和实际计费重，直接输出边界重量以及空运/海运结论。

## 海运机会成本

海运机会成本是某个SKU占用40HQ稀缺体积或重量容量的价值，不一定是单独发生的货代账单：

```text
海运机会成本
= 40HQ总成本 × MAX（SKU体积 ÷ 有效装载体积，SKU实重 ÷ 有效装载重量）
```

- 整柜长期满载：新品会挤掉其他货或更早触发下一柜，应使用占柜机会成本
- 柜内已有确定空位、放不放该SKU都不会改变整柜费用：干线边际成本可设为0
- 一组新品会导致额外开柜：单SKU线性分摊后，还需做一次整组装柜校正

## 功能

- 输入每个 SKU 的产品点和有效提前天数
- 计算实重、体积重、计费重和抛重倍数
- 支持 40HQ 体积/重量占柜机会成本
- 输出 AAI 排名、成功情景盈亏平衡点数和提前天数
- 按空运增量预算或最多 SKU 数给出组合
- 支持 JSON 导入导出、CSV 结果导出、打印和本机自动保存
- 单文件 HTML，无后端，录入数据不会上传服务器
