# 新品空运优先级计算器

用于首批测款时，按“每 1 元空运新增成本能够提前释放多少毛利潜力”排列 SKU 的空运优先级，并计算每个产品的空运/海运边界计费重。

## 核算口径

公司原本就会持续发整柜，不代表测款 SKU 的海运成本为零。本工具采用完整成本分摊口径：将每个 40HQ 约 US$8,000 的整体总费用，按照 SKU 占用的体积或重量比例分摊给该 SKU。

默认取体积占比与重量占比中较高者，避免低估轻抛货或重货对柜内容量的占用：

```text
整柜人民币成本 = 40HQ整体总费用（美元）× 汇率
体积分摊比例 = SKU整批海运体积 ÷ 40HQ有效装载体积
重量分摊比例 = SKU整批海运毛重 ÷ 40HQ有效装载重量
海运分摊成本 = 整柜人民币成本 × MAX（体积分摊比例，重量分摊比例）
                 + 单SKU海运固定费用
```

US$8,000 是默认值，可按公司的实际整柜总成本调整。默认汇率为 6.8，因此默认整柜人民币成本为 ¥54,400。有效装载体积、有效装载重量、汇率和单 SKU 海运固定费用也都可以在页面中修改。

## 产品点口径

- 1 产品点 = 产品成功后年毛利潜力 ¥120,000
- 新增产品默认 0.1 点，即年毛利潜力 ¥12,000，可逐个修改
- 允许填写小数，例如 0.5 点代表年毛利潜力 ¥60,000
- 所有产品应使用公司统一的毛利定义

产品点表达的是产品做成大货后的长期毛利潜力。首次测款采购价可能与正常大货采购价差异很大，因此首次采购成本不直接进入产品点，也不直接决定 AAI 排名；它仍可作为实际现金预算或人工决策的补充条件。

## 核心公式

```text
年毛利潜力 = 产品点 × ¥120,000
提前毛利潜力 = 年毛利潜力 × 有效提前天数 ÷ 365

空运新增成本 = 空运成本 − 海运分摊成本
AAI = 提前毛利潜力 ÷ 空运新增成本
```

AAI 越高，说明同样多花 1 元选择空运，能够提前释放的毛利潜力越大，因此越应该排在空运队列前面。

默认优先级分档为：

- AAI ≥ 3：高优先
- 2 ≤ AAI < 3：中优先
- 1.5 ≤ AAI < 2：达到最低入选线，可纳入空运方案
- AAI < 1.5：未达到最低入选线，建议海运

高优先内部线与中优先内部线只负责给已经计算出的 AAI 分档，方便排序和阅读；它们不决定空海运边界。空海运边界只由“当前最低入选线”决定，默认使用 AAI=1.5。三条线都可以回调，修改最低入选线后，页面中的边界总成本、边界计费重和运输建议必须同步更新。

AAI 是成功情景下的优先级指标，不是经过成功率加权的财务收益率。若空运和海运不改变同一产品的测款成功率，成功率对两种运输方式相同，也对所有产品统一，则不需要把它加入主排名：统一乘数不会改变 SKU 的先后顺序。需要做组合期望值分析时，可以再用公司统一的测款成功率校准最终选择线。

## 重量、体积与空运成本

重量和体积会同时影响空运成本与海运分摊成本，因此都与最终结论有关。

```text
空运体积重 = 外箱长 × 宽 × 高 ÷ 抛重除数
空运计费重 = MAX（整批实重，整批体积重），再按货代规则进位
空运成本 = 空运计费重 × 空运单价 + 空运固定费用
```

轻抛货通常会因体积重提高空运计费重；重货则可能因重量占比较高而增加海运分摊成本。页面会自动比较两种运输方式，不应只用一个与体积、重量无关的固定公斤数作为所有产品的边界。

## 空海运边界计费重

边界表示“AAI 刚好达到当前最低入选线”的位置。设最低入选线为 `T`，则：

```text
边界允许的空运新增成本 = 提前毛利潜力 ÷ T
边界空运总成本 = 海运分摊成本 + 提前毛利潜力 ÷ T
空运边界计费重
=（边界空运总成本 − 空运固定费用）÷ 空运单价
```

- 实际空运总成本 ≤ 边界空运总成本，同时实际计费重 ≤ 边界计费重：达到最低入选线，优先考虑空运
- 实际空运总成本 > 边界空运总成本，同时实际计费重 > 边界计费重：未达到最低入选线，优先考虑海运

其中：

```text
实际空运总成本 = 实际空运计费重 × 空运单价 + 空运固定费用
```

成本比较与公斤数比较是同一判断的两种表达，不能只显示公斤数。结果区应明确展示实际空运总成本，并显示“实际空运总成本 ≤ / > 边界空运总成本”；公斤数区域同时显示“实际计费重 ≤ / > 边界计费重”。

以默认 0.1 产品点、提前 50 天、空运费 ¥40/kg 为例，提前毛利潜力为：

```text
¥12,000 × 50 ÷ 365 = ¥1,643.84
```

“基础边界（未计具体 SKU 海运分摊）”就是暂时把该 SKU 的海运分摊成本按 ¥0 计算，只用于给出一个不依赖具体体积和毛重的参考值；它不代表公司实际海运费为零。基础边界使用最低入选线，而不是固定使用 AAI=1。

按默认最低入选线 AAI=1.5、空运费 ¥40/kg、空运固定费用 ¥0 计算：

```text
基础边界允许的空运新增成本 = ¥1,643.84 ÷ 1.5 = ¥1,095.89
基础边界空运总成本 = ¥0 + ¥1,095.89 = ¥1,095.89
基础边界计费重 =（¥1,095.89 − ¥0）÷ ¥40/kg = 27.40kg
```

高优先线 3 和中优先线 2 与这个边界没有直接关系；只有最低入选线 1.5 进入上述公式。具体 SKU 的准确边界还要加上其自动计算的海运分摊成本，所以会随整批体积和整批毛重变化，并不都是 27.40kg。

使用边界模块的默认整批体积 0.1m³、整批毛重 10kg、40HQ 有效体积 68m³、有效载重 26,000kg 计算时，体积占比高于重量占比：

```text
整柜人民币成本 = US$8,000 × 6.8 = ¥54,400
海运分摊成本 = ¥54,400 × MAX（0.1 ÷ 68，10 ÷ 26,000）= ¥80.00
边界允许的空运新增成本 = ¥1,643.84 ÷ 1.5 = ¥1,095.89
边界空运总成本 = ¥1,095.89 + ¥80.00 = ¥1,175.89
边界计费重 = ¥1,175.89 ÷ ¥40/kg = 29.40kg
```

在默认空运抛重除数 6,000、按 0.5kg 向上进位时，同一批货的实际空运计费重也会由页面自动填写：

```text
体积重 = 0.1m³ × 1,000,000 ÷ 6,000 = 16.67kg
实重与体积重取较高者，再按 0.5kg 向上进位 = 17.00kg
```

因此默认示例中，实际空运总成本为 `17.00kg × ¥40/kg + ¥0 = ¥680.00`，低于边界空运总成本 ¥1,175.89；实际计费重 17.00kg 也低于边界 29.40kg，达到默认最低入选线 AAI=1.5，建议空运。用户仍可直接覆盖自动填写的实际计费重，以货代给出的最终计费重为准；修改体积、毛重、抛重除数或进位单位后，自动值会随之更新。

若使用截图中的空运单价 ¥45/kg、固定费用 ¥10，实际计费重仍为 17.00kg，则页面应展示：

```text
实际空运总成本 = 17.00kg × ¥45/kg + ¥10 = ¥775.00
边界空运总成本 = ¥1,095.89 + ¥80.00 = ¥1,175.89
边界计费重 =（¥1,175.89 − ¥10）÷ ¥45/kg = 25.91kg
成本比较：¥775.00 < ¥1,175.89
公斤数比较：17.00kg < 25.91kg
结论：达到最低入选线 AAI=1.5，建议空运
```

页面提供独立的“边界计费重快速计算”模块。输入产品点、有效提前天数、整批海运体积和整批海运毛重后，模块会自动继承全局的整柜费用、汇率、有效装载容量、空运抛重规则与运价，并自动计算实际空运计费重。用户也可以手工覆盖该值。模块输出：

- 海运分摊成本及采用的分摊依据
- 对应当前最低入选线（默认 AAI=1.5）的空运总成本与边界计费重
- 当前实际空运总成本，以及它与边界空运总成本的大小比较
- 当前实际计费重对应的空运或海运建议

右侧结果区的每个指标都带有计算说明。电脑端将鼠标悬停在指标卡片上即可即时查看；键盘操作时将焦点移入卡片也会显示相同说明，便于核对取值来源和公式。

## 功能

- 输入每个 SKU 的产品点和有效提前天数（默认 50 天）
- 计算实重、体积重、计费重和抛重倍数
- 按体积占比或重量占比自动分摊 40HQ 完整成本
- 输出 AAI 排名、成功情景盈亏平衡点数和提前天数
- 为每个 SKU 计算独立的空运边界计费重
- 按空运新增预算或最多 SKU 数给出组合
- 支持 JSON 导入导出、CSV 结果导出、打印和本机自动保存
- 单文件 HTML，无后端，录入数据不会上传服务器
